手動でシグナリングするp2pのサンプルです。

```bash
python ./../../../python/src/server/server.py

# 1. 以下をブラウザAとブラウザBで開く
# http://localhost:8000/

# 2. ブラウザAの[start]ボタンをクリック
# 3. ブラウザAのConsoleにsdpが表示されるのでブラウザBのテキストボックスにコピペする。この時最後の行に改行を入れること。
# 4. ブラウザBの[set]ボタンをクリック
# 5. ブラウザBのConsoleにsdpが表示されるのでブラウザAのテキストボックスにコピペする。この時最後の行に改行を入れること。
# 6. ブラウザAの[set]ボタンをクリック
# 7. 片方のブラウザの[send]ボタンをクリックするともう片方のブラウザのConsole上に時刻が表示される。
```
